===============================================================================
                             CPUIdleStabilizer
===============================================================================

Prevents CPU Idle Instability by Maintaining a Low, Steady Load.

-------------------------------------------------------------------------------
[ INSTALLATION & SETUP ]
-------------------------------------------------------------------------------

There are two versions of this application available. Please follow the 
instructions for the version you have downloaded:

1. STANDALONE VERSION (CPUIdleStabilizer_Standalone_vX.X.X.zip)
   ------------------------------------------------------------
   * Best for: Most users who want a "plug and play" experience.
   * Requirements: None (Includes the .NET runtime).
   * Instructions:
     1. Extract the ZIP file to a folder of your choice.
     2. Run 'CPUIdleStabilizer.exe'.

2. LIGHTWEIGHT VERSION (CPUIdleStabilizer_Lightweight_vX.X.X.zip)
   ------------------------------------------------------------
   * Best for: Users who already have .NET 8 installed and want a tiny download.
   * Requirements: .NET 8 Desktop Runtime.
   * Download .NET 8: https://dotnet.microsoft.com/en-us/download/dotnet/8.0
   * Instructions:
     1. Ensure the .NET 8 Desktop Runtime is installed.
     2. Extract the ZIP file to a folder of your choice.
     3. Run 'CPUIdleStabilizer.exe'.

-------------------------------------------------------------------------------
[ QUICK START GUIDE ]
-------------------------------------------------------------------------------

1. Click the 'START' button to begin stabilization. The status bar will turn
   green to indicate active load generation.
2. Recommended Settings:
   - Target CPU Load: 3% - 5% (Adjust the slider).
   - Eco Mode: ON (Recommended to prevent resonance).
3. Click 'Hide to Tray' or close the window to keep the app running silently 
   in the system tray.
